## üöÄ Overview

**DataVector** is a powerful, client-side web application designed to handle all your document processing needs securely within your browser. From converting PDFs to editing and compressing files, everything happens locally on your device‚Äîensuring your sensitive data never leaves your control.

## üõ†Ô∏è Key Features

### üìÑ PDF Tools
| Feature | Description |
| :--- | :--- |
| **Merge PDF** | Combine multiple PDFs into a single document with drag-and-drop reordering. |
| **Split PDF** | Extract specific pages or split documents into individual files. |
| **Compress PDF** | Reduce file size with adjustable quality slider and real-time size estimation. |
| **PDF to JPG** | Convert PDF pages into high-quality images. |

### üîÑ Converters
- **Word to PDF**: Convert `.docx` and `.doc` files instantly.
- **Excel to PDF**: Transform spreadsheets `.xlsx` into clean PDFs.
- **PPT to PDF**: Render PowerPoint presentations `.pptx` as PDFs (Client-side rendering!).
- **HTML to PDF**: Convert HTML files or code snippets to portable documents.

### üì∑ Scanning & Utility
- **Scan to PDF**: Capture documents using your device camera.
- **DocFix**: Resize and format photos/signatures for passports and online forms.
- **Search**: Built-in search engine for open directories and web resources.

---

## üíª Tech Stack

- **Frontend**: HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+)
- **Libraries**:
  - `pdf-lib` & `pdf.js` - PDF manipulation
  - `mammoth.js` - Word conversion
  - `SheetJS` (xlsx) - Excel processing
  - `PPTXjs` - PowerPoint rendering
  - `html2pdf.js` - HTML conversion
  - `Sortable.js` - Drag-and-drop interfaces
- **Deployment**: Ready for Cloudflare Pages / Static Hosting
